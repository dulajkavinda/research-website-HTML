  • C. Callison-Burch and P. Shilane, "A Natural Language Question and Answer System," American Journal of Computational Linguistic, vol. 8, pp. 110-122.

    A Natural Language Question Answer System is proposed by C. Callison-Burch and P. Shilane. It is a grammar-based system which maps English questions and statements into predicate logic. User input is parsed and converted to KIF (Knowledge Interchange Format) and is sent to a reasoner that does interpretation using a set of first order logic axioms and returns a solution. System implements the Ginsburg and Sag (forthcoming) theory of English interrogative constructions. Creating a correspondence between the syntax and semantics of a language allows the surface form of statements to be mapped onto a more abstract, logical representation. The program does cover a wide range of questions, it is lacking an analysis for some questions.

  • O. Ferret, B. Grau, M. Hurault-Plantet, G. Illouz, C. Jacquemin and L. Monceaux, "How NLP Can Improve Question Answering," KNOWLEDGE ORGANIZATION, vol. 29, no. 3-4, pp. 135-155, 2002.

    Answering open-domain factual questions requires Natural Language Processing for refining document selection and answer identification. This performs an analysis of documents relying on multi-word term search and their linguistic variation both to minimize the number of documents selected and to provide additional clues when comparing question and sentence representations. Question analysis is conducted with the two following objectives: the first is to extract some characteristics which will likely be used in the answer extraction module; the second is to extract the terms which will re-index the selected documents in order to retain only a subset of them and to supply further evidences during the final matching. They have already underlined the importance of the parsing of the question to find the expected answer type. As done by the system, most systems first determine the expected answer type by seeking pre-defined patterns in the question. So, they used it for 400 different patterns to identify about 50 answer types.
